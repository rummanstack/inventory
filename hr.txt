# IMPLEMENT A PRODUCTION-READY HR MODULE

You have full access to this codebase and have already analyzed the project.

Your task is to replace the existing basic HR module with a professional HR module that integrates seamlessly into the current ERP.

## VERY IMPORTANT

This is **NOT** a standalone HR system.

This HR module must become a native part of the existing ERP.

Do NOT redesign the ERP.

Do NOT change existing architecture.

Do NOT duplicate existing code.

Always extend existing functionality whenever possible.

---

# First Task (MANDATORY)

Before writing any code:

1. Analyze the current HR implementation.
2. Identify all existing HR-related tables.
3. Identify all HR APIs.
4. Identify all HR pages.
5. Identify reusable services.
6. Identify reusable components.
7. Identify existing permissions.
8. Identify accounting integration points.

Generate a short implementation plan.

Only after understanding the current implementation should development begin.
